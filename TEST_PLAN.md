## 기능 테스트 항목

### Backend API 테스트

#### Health Check API
- [x] GET /api/health - 정상 응답 확인
- [x] 응답 형식 검증 (status: "ok", message 포함)

#### Memo API
- [x] POST /api/memos - 메모 생성
  - 제목 필수 입력 검증
  - 내용 선택 입력 검증
  - 생성된 메모 ID 반환 확인
- [x] GET /api/memos - 메모 리스트 조회
  - 빈 리스트 반환 확인
  - 여러 메모 정렬 확인 (최신순)
- [x] DELETE /api/memos/{id} - 메모 삭제
  - 존재하는 메모 삭제 확인
  - 존재하지 않는 메모 삭제 시 404 에러 확인
